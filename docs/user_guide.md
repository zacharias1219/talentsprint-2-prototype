# User Guide

## Getting Started

Welcome to the AI-Powered Financial Advisor! This guide will help you get started.

## Features

### 1. User Profile Creation
- Create your profile with demographics and risk tolerance
- Update your profile as your situation changes

### 2. Chat Interface
- Ask questions about investments, retirement planning, savings, etc.
- Get personalized financial advice

### 3. Recommendations
- Receive personalized investment recommendations
- View portfolio allocation suggestions
- See risk-return analysis

## Using the Chat Interface

1. Enter your User ID in the sidebar
2. Type your question in the chat input
3. Receive AI-generated responses with sources
4. Review disclaimers and important information

## Best Practices

- Provide accurate information in your profile
- Review recommendations carefully
- Consult with a qualified financial advisor for major decisions
- Keep your profile updated

## Support

For issues or questions, refer to the project documentation or contact support.
